## Japanease

**使い方:** ファイルを読込むだけで、ページ内リンクをスムーズに移動するようにできます。ページのトップに戻る場合は href="#pagetop" としてしてください。プラグイン本体の初期値を変更すると、アニメーションの間隔やアンメーションの種類を変更できます。
